Events in Null Recipes are a way to keep track of all the important happenings in your life! When creating or editing an event, make sure that you:
  • Include a name for your event; this will be shown in the My Event section of your Profile Page.
  • Include a description for your event so you don't forget what the event entails.
  • Include a date for your event so you don't miss it!
